# Instagram Bot

> Instagram Bot created for SoftTech 2017 Hackathon Instagram Award. Takes images from the camera with using OpenCV at certain intervals and uploads them to Instagram

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

## Prerequisites

-  [LevPasha Instagram API](https://github.com/LevPasha/Instagram-API-python.git)
-  [OpenCV 2.8.4](http://opencv.org/opencv-2-4-8.html)


## Usage

- Install the dependencies.
- Set user name. Change 71. line with your user name and password.
```sh
 $71 instagramAPI = InstagramAPI("USER NAME", "PASSWORD")
 ```
- Set time interval from line 75
```sh
 $75 timeInterval = 2
 ```
